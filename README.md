# 🎬 Movie Recommendation System

A content based movie recommendation system built using the TMDB 5000 dataset. It suggests similar movies based on content such as genres, keywords, cast, and overview.

---

## 🚀 Features
- Recommends movies similar to a selected movie  
- Content-based filtering approach  
- Fast and simple recommendations  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- NLTK  

---

## 📂 Dataset
- TMDB 5000 Movie Dataset

**Dataset link**  
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata/data

---

## ⚙️ How It Works
- Data cleaning and preprocessing  
- Feature extraction 
- Text processing using NLP techniques  
- Vectorization of text data  
- Similarity calculation between movies  
- Recommendation of top similar movies
